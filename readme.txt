== Guide de démarrage ==
------------------------
1) Installation:
    -langages:
	-> python3
	-> HTML5
    -packages:
	-> Flask (version 1.1.2)
	-> SQLAlchemy (version 1.4.11)
2) Ouvrir et run main.py
3) Ouvrir l'adresse qui apparaît dans la console dans le navigateur

== Arboresence des fichiers ==
------------------------------
/Projet2
=> création de la base de données dans db_create.py
=> données fournis pour le projet dans 1002-sql-data
/Projet2/Site_Web
=> démarrage du site dans main.py
/Projet2/Site_Web/website
=> database générée par db_create.py dans DataFerme.db
=> routes du site et extraction des données SQL dans views.py
/Projet2/Site_Web/website/static
=> image de la page principale du site vaches.jpg
=> style CSS du site dans main.css
/Projet2/Site_Web/website/static/templates
=> base du site dans base.html
=> page d'accueil du site dans home.html
=> figure 1 dans question1.html
=> figure 2 dans question2.html
=> figure 3 dans question3.html
=> figure 4 dans question4.html
=> figure 7 dans question7.html
